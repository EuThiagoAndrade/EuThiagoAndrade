Este perfil é meu depósito público de experimentos, automações, estudos e pequenos sistemas. Alguns nasceram de uma necessidade prática. Outros, da vontade de entender melhor uma ideia. Se forem úteis para mais alguém, melhor ainda.

## O que tem por aqui

Nada de grande manifesto técnico. Cada repositório explica sua própria bagunça, seu propósito e suas tecnologias.
